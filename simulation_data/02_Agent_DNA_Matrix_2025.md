
# 02. 参演企业智能体 DNA 矩阵 (2025版)

**基准时间**: 2025-11-22
**状态更新**: 根据2024-2025年实际发展调整了部分参数。

---

## 1. 蓝箭航天 (LandSpace)
*   **战略原型**: **绝对领跑者 (The Frontrunner)**
*   **当前状态**:
    *   `Tech`: 朱雀三号 (ZQ-3) 处于入轨回收试验前夕。
    *   `Cash`: 资金充裕，主要用于发射场建设和产能扩充。
*   **DNA特征**:
    *   **风险偏好**: 中 (从激进转为稳健，保成功率)
    *   **技术路线**: 液氧甲烷 + 不锈钢 (全行业唯一)
    *   **对策逻辑**: 利用先发优势，挤压对手的融资空间和发射窗口。

## 2. 星河动力 (Galactic Energy)
*   **战略原型**: **双模驱动者 (Dual-Mode Driver)**
*   **当前状态**:
    *   `Cash Cow`: 谷神星一号 (固体) 保持高频发射，提供稳定现金流。
    *   `R&D`: 智神星一号 (液体) 正在进行关键试车。
*   **DNA特征**:
    *   **风险偏好**: 低 (高度重视商业闭环)
    *   **技术路线**: 固体现有 + 液氧煤油研发
    *   **对策逻辑**: "以固养液"，确保在2027年不掉队，同时维持上市财务指标。

## 3. 深蓝航天 (Deep Blue)
*   **战略原型**: **绝地求生者 (Survival Techie)**
*   **当前状态**:
    *   `Focus`: 全力攻关星云-1号的入轨回收，这是其生存的唯一筹码。
*   **DNA特征**:
    *   **风险偏好**: 极高 (All-in 回收技术)
    *   **技术路线**: 液氧煤油 + VTVL
    *   **对策逻辑**: 必须在2026年前完成回收，否则面临被并购风险。对政府“首台套”补贴高度敏感。

## 4. 中科宇航 (CAS Space)
*   **战略原型**: **体系依托者 (System Integrator)**
*   **当前状态**:
    *   `Market`: 依托中科院卫星订单，力箭系列发射稳定。
    *   `New Biz`: 开始布局太空旅游试验飞行。
*   **DNA特征**:
    *   **风险偏好**: 低
    *   **技术路线**: 固体大推力 + 液体可回收 + 太空旅游吊舱
    *   **对策逻辑**: 响应政府号召，建设共性平台，不仅卖火箭，更卖“服务”。

## 5. 星际荣耀 (i-Space)
*   **战略原型**: **技术迭代者 (Tech Iterator)**
*   **当前状态**:
    *   `Pivot`: 从早期的固体失利中恢复，全力押注双曲线三号。
*   **DNA特征**:
    *   **风险偏好**: 中高
    *   **技术路线**: 液氧甲烷 (焦点发动机)
    *   **对策逻辑**: 试图在技术指标（如复用次数）上超越蓝箭，以差异化参数争取军方/政府订单。
